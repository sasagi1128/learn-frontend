# React Learning Plan — 4-Week Frontend Roadmap

## Goal
By the end of Week 5, I will have a working React frontend that connects to my REST API with authentication.

## Week 1 — Learn Front End 2: React Fundamentals
**Goal:** Get comfortable with React components, JSX, and useState.
**Resources:** https://react.dev/learn/tutorial-tic-tac-toe
**Tasks:** Create components, use useState, push to GitHub

## Week 2 — Learn Front End 3: Fetching Data from an API
**Goal:** Use useEffect and fetch() to load data from an API and display it.
**Resources:** https://react.dev/learn/synchronizing-with-effects, https://reactrouter.com
**Tasks:** Fetch from public API, display a list, add React Router

## Week 3 — Learn Front End 4: Authentication
**Goal:** Build a login form, send credentials to the API, store JWT token, protect routes.
**Resources:** https://reactrouter.com/en/main/start/examples
**Tasks:** POST to login endpoint, store token, make authenticated requests

## Week 4 — Learn Front End 5: Full API Integration
**Goal:** Complete frontend connected to all major REST API endpoints.
**Resources:** https://tailwindcss.com/docs, https://react.dev/learn/scaling-up-with-reducer-and-context
**Tasks:** Full CRUD with API, logout button, polish UI, end-to-end test
